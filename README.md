# Dungeons & Habits Tracker

## Main Idea

Dungeons & Habits Tracker is a small RPG-inspired habit tracker. 

Right now I'm stating as a CLI prototype where the users can create characters, assigns attributes, and manage habits.

## Core Idea

The main concept is to connect everyday actions to character growth.

- `Body` represents physical habits like workouts, movement, and health
- `Mind` represents studying, focus, and learning
- `Social` represents communication, relationships, and social effort

The long-term goal is for real-world consistency to shape in-game progression in a simple and motivating way.

## Future Direction

Future versions of the project will include:

- habit logging and progress tracking
- experience, leveling, and skill growth
- inventory, items, and gameplay systems
- persistence with a database
- a web or mobile interface
